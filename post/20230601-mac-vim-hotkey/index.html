mac下vim 删除全部的方式

2023-06-01

mac 下 vim 删除全部的方式：

在 Mac 的 vim 编辑器中，如果要删除全部内容，可以使用以下命令：

  1. 进入 vim 编辑器后，按下 “Esc” 键，确保处于命令模式下。

  2. 输入以下命令： gg dG 其中，gg 是跳转到文件的第一行，dG 是删除从当前行到文件的最后一行。 (注意大小写)

  3. 按下回车键，编辑器将删除整个文件的内容。

请注意，这将无法恢复地删除文件中的所有内容，因此请确保在执行此操作之前已备份文件内容。
